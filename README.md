# Projeto – Azure Functions + APIs Serverless | Bootcamp DIO

Este repositório contém a implementação prática desenvolvida para o desafio do bootcamp da Digital Innovation One (DIO).  
O objetivo do projeto é explorar o uso do **Azure Functions** para criar uma **API Serverless**, aplicando conceitos fundamentais de computação em nuvem e boas práticas de desenvolvimento.

## Objetivo do Projeto

- Criar uma Function App no Azure usando **HTTP Trigger**
- Construir uma API Serverless simples e escalável
- Explorar conceitos de Serverless, gatilhos (Triggers) e bindings
- Estruturar um repositório profissional para portfólio
- Documentar todas as etapas do projeto, melhorando a apresentação técnica

## Tecnologias Utilizadas

- **Azure Functions**
- **Azure Portal**
- **Azure Storage Account**
- **.NET / C#** (ou Python/JavaScript, conforme sua escolha)
- **Git & GitHub**
- **VS Code + Azure Functions Extension**
